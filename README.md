# Salon Booking Web Application

This is a full-stack ASP.NET Core MVC web application designed for a modern salon. 
Customers can browse services, Search and view salon products, and make bookings. 
Staff can log in to manage bookings and perform administrative tasks.

The system is built using:

- **ASP.NET Core MVC**
- **C# and Razor Views**
- **Entity Framework Core (Code First)**
- **SQL Server (LocalDB)**
- **HTML5, CSS3, JavaScript**
- **jQuery & AJAX**
